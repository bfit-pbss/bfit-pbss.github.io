Bfit-Public Bicycle Sharing System

Bfit is a system, or a service in which bicycles are made available for shared use to individuals on a short term basis for a price. Bike share system allow people to borrow a bike from a "dock" and return it at another dock belonging to the same system. Docks are special bike racks that lock the bike, and only release it by computer control. The user enters their payment information, and the computer unlocks a bike. When the user returns the bike, they place it in the dock, which locks it into the dock.
